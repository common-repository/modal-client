Plugin created to add a modal to the website. This modal has options so the client can choose where to go.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/modal-client` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Modal Client screen to configure what should appear on each modal
4. On the page of your preference insert the shortcode '[modal-client]' to add the button that will open the modal
